¿Qué es una cadena de Markov?

En terminos tecnicos podemos resumir que una cadena de markov es un grafo probabilistico. Para la gente que esta contextualizada este termino no les hara mucho sentido asi que procederemos a explicarlo.

Un grafo es una coleccion de nodos y aristias, sirven para representar varias cosas de la vida real como ciudades donde cada nodo es un ciudad y cada arista la distancia entre cada una, ojo que este es un grafo ponderada ya que el peso de cada arista sera diferente este representando la distancia que hay de una ciudad a otra

Un grafo probabilistico entonces seran nodos que estos (por ejemplo) pueden representar estados de un objeto, liquido, solido y gaseoso, los cuales podemos conectar mediante aristas. Donde estas aristas tendrian de peso la probabilidad que este elemento pase de un estado a otro, por ejemplo de liquido a gaseoso.

¿Qué hace esta haciendo esta página?

Podemos simplificar el proceso en dos partes: crea una cadena de markov y genera otra a partir de esa cadena

Para generar una cadena de markov necesitamos datos previos para empezar a poder crear nuestro grafo, en este caso un imagen. Extraemos todos los pixeles y a partir de ellos podemos obtener la probailidad de que haya X pixel despues de Y recorriendolos todos mientras registramos frequencias.

Por ultimo generamos una nueva imagen. Elegimos un pixel aleatoriamente y a partir de ahi empezamos a recorrer nuestro grafo con una caminata aleatoria siguiendo las probabilidades que fueron asignadas a cada pixel. Hacemos el proceso para 250,000 pixeles para generar una imagen de 500x500.

¿Por qué salio la imagen asi?

Entonces lo unico que queda por responder es, ¿Si todo esto tiene una relacion de cada pixel entre cada uno porque no sale una imagen con objetos mas claro? Lo que pasa es que a diferencia de nosotros un grafo probabilisitico no es inteligente.

Nosotros conocemos el significado de los pixeles y las formas que crean cuando estan juntos, pero para una maquina o una cadena de markov son simplemente otra serie mas de pixeles que tiene que interpretas, ellos nos buscan encontrarle sentido.

Por ultimo entonces el resultado que tenemos al final es una reogranizacion de los pixeles que tenemos, demostrando los colores dominantes y como una cierta interpretacion de que solo nosotros le damos sentido a una serie de pixeles.
